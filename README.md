# Wi-Fi-Audio-Demo
A demo for Wi-Fi Audio based on nRF5340 Audio DK + nRF7002 EK

# Hardware
Recorder: nRF5340 Audio DK + nRF7002 EK

Player: nRF5340 Audio DK + nRF7002 EK / UDP+Opus client on PC and Mobilephone
